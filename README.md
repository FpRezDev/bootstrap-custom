# bootstrap-custom
Project template for creating custom bootstrap projects

Screenshot
---------

![alt text](/screenshot.png "bootstrap-custom screenshot")

Install
---------

```
git clone https://github.com/FpRezDev/bootstrap-custom.git
```

Build Css/Js/Libs
---------

```
npm run build
```

Available scripts
---------

```
npm run build
npm run buildCss
npm run buildJs
npm run buildLibs
npm run clean
npm run cleanCss
npm run cleanJs
npm run cleanLibs
```
